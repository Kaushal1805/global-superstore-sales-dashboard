## 🔍 Key Business Insights

### 1. Category Performance
- Technology leads with **37.53%** of total revenue ($4.74M)
- Office Supplies contributes **32.52%** ($4.11M)
- Furniture accounts for **29.96%** ($3.79M)

### 2. Geographic Insights
- **APAC** is largest market with **$3.59M** revenue
- **USA** is top country with **$2.30M** revenue
- **Australia** ranks 2nd with **$0.93M**

### 3. Product Insights
- **Copiers** most profitable sub-category ($0.26M profit)
- **Phones** 2nd highest profit ($0.22M)
- **Tables** show consistent losses (-0.1M profit)

### 4. Customer Insights
- **Consumer segment** dominates with **51.48%** sales
- **Tom Ashbrook** is top customer ($31K revenue)
- **Medium priority** orders = 57.25% of all orders

### 5. Discount Impact
- Higher discounts directly reduce profit
- Discount above 20% leads to negative profit

### 6. Shipping Analysis
- **Standard Class** highest usage ($0.39M cost)
- **Same Day** shipping most expensive ($0.07M)

## 🛠️ Tools Used
- **Power BI Desktop** - Dashboard creation
- **Power Query** - Data cleaning & transformation
- **DAX** - Custom measures & calculations

---

## 📂 Dataset Info
- **Source:** Global Superstore Dataset
- **Records:** 51,290 transactions
- **Period:** 2011 - 2014
- **Markets:** APAC, EU, US, LATAM, EMEA, Africa, Canada
- **Countries:** 147 countries

---

## 🧹 Data Cleaning Steps
1. Removed **Postal Code** column (80% null values)
2. Set correct data types for Date columns
3. Created calculated measures using DAX
4. Handled geographic data categorization
